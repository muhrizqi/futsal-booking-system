🔑 IMPORTANT NOTES
==================

- Copy .env.example → .env
- Edit .env dengan password yang aman
- Ganti WHATSAPP_ADMIN_JOGO & ADMIN_4R
- Change ADMIN passwords dari default

📊 DATABASE TABLES (Auto-created)
- venues (lokasi futsal)
- courts (lapangan)
- bookings (reservasi)
- pricing (harga per jam)
- audit_logs (activity logs)

🔐 SECURITY
- No hardcoded passwords
- Environment variables untuk secrets
- SQL injection prevention
- CORS configured
- Audit logging

💾 BACKUP STRATEGY
- Automatic daily backup
- Database + files backup
- Keep 7 latest backups
- Easy restore process
- No data loss


📈 SPECIFICATIONS
=================

Technology Stack:
  Frontend:      HTML5, CSS3, Vanilla JavaScript
  Backend:       Node.js 16+, Express.js
  Database:      PostgreSQL 13+
  Deployment:    Docker, PM2, systemd, Easy Panel
  OS:            Raspberry Pi OS, Linux

Performance:
  Load time:     < 100ms
  API response:  < 50ms
  Memory usage:  ~50MB
  Supports:      Unlimited courts & bookings

System Requirements:
  Minimum:  Raspberry Pi 2B+ (1GB RAM)
  Recommended: Raspberry Pi 4B (2GB+ RAM)
  Storage:  500MB + database size
